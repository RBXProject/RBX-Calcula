# Correção da Logo RBX no GitHub Pages

Nesta versão, a logo principal da RBX foi embutida diretamente no index.html.

## Como corrigir no GitHub

1. Extraia este ZIP.
2. No GitHub, clique em Add file > Upload files.
3. Envie os arquivos extraídos novamente para a raiz do repositório.
4. Confirme em Commit changes.
5. Aguarde o GitHub Pages republicar.

Também funciona substituir apenas o index.html, mas o recomendado é enviar todos os arquivos extraídos.
